# Curso de Analista de Dados - Big Data com Python

Este repositório contém informações e projetos desenvolvidos durante o curso de Big Data, utilizando Python para análise de dados. O curso abordou todas as etapas fundamentais de programação, incluindo:

- Variáveis
- Estruturas de repetição
- Condições
- Listas
- Séries
- Criação de DataFrames

Além disso, aprendemos a criar e consultar bancos de dados no SQL, utilizando modelos conceituais de entidades e relacionamentos para estruturar os dados.

## 📚 Principais bibliotecas utilizadas

- **Pandas**: para manipulação e análise de dados de forma eficiente.
- **NumPy**: para operações matemáticas e manipulação de arrays.
- **OpenPyXL** e **xlrd**: para trabalhar com arquivos Excel.
- **Matplotlib (pyplot)**: para criação de gráficos e dashboards diretamente no Python.

## 📊 Conceitos de Estatística

O curso também incluiu conceitos de estatística, como:

- Cálculo de média, mediana e quartis (1º, 2º e 3º)
- Amplitude
- Limites inferior e superior
- Identificação de outliers
- Outros conceitos estatísticos importantes

## 🛠️ Relatórios e Projeto Final

Durante o curso, foram desenvolvidos diversos relatórios no **Power BI**, integrando as análises realizadas para trabalhar com Big Data de forma visual e interativa.

### Projeto Final: **Data Analytics Aplicado à Segurança**

O projeto final consistiu na elaboração de um relatório analítico sobre **Homicídios Dolosos no estado do Rio de Janeiro**, utilizando o **Power BI**. O estudo de caso incluiu:

- Desenvolvimento de um **dashboard analítico** focado na análise de dados de segurança pública.
- Tratamento de grandes volumes de dados brutos com a **Linguagem M**.
- Conversão de strings temporais em séries históricas precisas.
- Modelagem avançada em **DAX**, com aplicação da metodologia estatística de **Tukey (Amplitude Interquartil - IQR)** para identificação dinâmica de outliers por **Área Integrada de Segurança Pública (AISP)**.
- Diferenciação de variações sazonais e desvios críticos, oferecendo uma visão clara e baseada em evidências das regiões que operam fora da normalidade estatística.
